# Quotes App

A simple and interactive **Random Quotes Generator** built using **HTML, CSS, and JavaScript**.
The app displays a random quote with its author every time the user clicks the "New Quote" button.

 
## Features

* Generate a random quote every click
* Display quote author name
* Fast and lightweight
* Responsive design using Bootstrap
* Simple and clean user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap
* Font Awesome

## How It Works

* Quotes are stored inside a JavaScript array.
* When the user clicks the "New Quote" button:

  * JavaScript generates a random number.
  * The quote and author are displayed dynamically.
